1. angular.json
Stores the Angular project configuration, including build, serve, assets, styles, and production settings.

2. tsconfig.json
Contains the base TypeScript compiler configuration for the project.

3. tsconfig.app.json
Contains the TypeScript compiler configuration for the Angular application.

4. package.json
Contains project metadata, dependencies, and npm scripts.

5. src/main.ts
The entry point of the Angular application that bootstraps the root application.

6. src/app/app.config.ts
Contains the configuration for the standalone Angular application.

7. src/app/app.ts
Defines the root component of the Angular application.

8. src/index.html
The main HTML page that hosts the Angular application.

9. Production Budget Configuration

maximumWarning:
Displays a warning if the application bundle size exceeds the specified limit (500 kB).

maximumError:
Fails the production build if the application bundle size exceeds the specified limit (1 MB).

anyComponentStyle:
Displays a warning if a component's CSS file exceeds 4 kB and fails the build if it exceeds 8 kB.
